## Getting Started

1. Install **Lightshot** and run it in the background.  
2. Press **Print Screen** (or your custom hotkey) to activate the capture tool.  
3. Select the desired area on your screen.  
4. Edit the screenshot using the on-screen toolbar.  
5. Save locally, copy to clipboard, or upload to generate a shareable link.  
6. Manage your uploaded screenshots via your personal gallery.
